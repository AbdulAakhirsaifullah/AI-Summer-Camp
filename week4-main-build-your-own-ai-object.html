Train it
AI learns from examples of "if this happens → do that." We'll give you 6 quick training situations — you guess what it does each time before we tell you.

3
Prompt it
All those "if → do" examples get turned into one instruction sheet called a prompt. Now ask it 3 test questions — each one is a situation you already trained it on in Step 2, so you guess the reply first!
✅ Prompt testing complete! It answered all 3 questions correctly.
4
Break it
Now ask about something it was never trained on. First guess: will it handle it, or get confused?
