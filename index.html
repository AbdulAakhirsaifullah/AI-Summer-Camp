About the project

One curriculum, two age-tuned pathways

This project contains two complete AI learning pathways designed for different age groups — Track A (8–11) and Track B (12–15). Each activity is interactive, browser-based, and requires no installation.

Every activity runs as a single, self-contained page — no accounts, no backend, no setup. Just open it and start exploring how AI actually works, one hands-on experiment at a time.
